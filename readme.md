Створено систему керування замовленнями, цей репозиторій містить ассаймент з проєктування бази даних.
Структура бази даних: (5 таблиць)
`customers` — дані клієнтів.
`products` — каталог товарів.
`orders` — інформація про замовлення.
* `order_items` — складові замовлення.
`order_log` — журнал для логування дій.
Виконані завдання: створено функцію, дві процедури та два тригери, також у кінці знаходиться тестування.
